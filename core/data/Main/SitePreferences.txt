---++ Modell Aachen Settings

Bereiche:
   * Set Entwicklung = Entwicklungsbereich
   * Set Main = Allgemeiner Bereich
   * Set WorkflowManagement = Interaktive Workflows
Templates:
   * Set ArbeitsanweisungTemplate = Arbeitsanweisung
   * Set ChecklisteTemplate = Checkliste
   * Set InfoseiteTemplate = Infoseite
   * Set ProzessbeschreibungTemplate = Prozessbeschreibung
Workflow:
   * Set WORKFLOW = 

---++ Appearance

Customize logo and tooltip texts:
   * Set NOAUTOLINK = on
   * Set SKIN = project,handbuch,modac
   * Set WIKILOGOIMG = %PUBURL%/%SYSTEMWEB%/ModacSkinTheme/ModacSkinLogos/qwiki-logo_small.png
   * Set WIKILOGOALT = Powered by Foswiki, The Free and Open Source Wiki
   * Set WIKILOGOURL = %SCRIPTURL{"view"}%/%USERSWEB%/%HOMETOPIC%
Give your site a specific name:
   * Set FOSWIKI_LAYOUT_URL = %PUBURL%/%SYSTEMWEB%/ModacSkinTheme/layout.css
   * Set FOSWIKI_STYLE_URL = %PUBURL%/%SYSTEMWEB%/ModacSkinTheme/style.css
   * Set FOSWIKI_COLORS_URL = %PUBURL%/%SYSTEMWEB%/ModacSkinTheme/colors.css
   * Set WIKITOOLNAME = Q.Wiki


Project-specific settings:
   * Set USERSTYLEURL = /pub/System/ModacSkinTheme/handbuch.css
   * Set TOC_TITLE = Inhaltsverzeichnis

Format of new links

   * Set NEWLINKFORMAT = <a class="foswikiNewLink" href="%SCRIPTURLPATH{"view"}%/$web/WebCreateNewTopic?parenttopic=%WEB%.%TOPIC%;newtopic=$topic" rel="nofollow" title="%MAKETEXT{"Create this topic"}%">$text</a>
Give your site a specific name:

   * Set WIKITOOLNAME = Q.Wiki
Customize your [[http://en.wikipedia.org/wiki/Favicon][favicon]]:

   * Set FAVICON = %PUBURL%/%SYSTEMWEB%/ModacSkinTheme/ModacSkinLogos/favicon.ico
Customize your top bar background image and color
   * Set WEBHEADERART = %PUBURLPATH%/%SYSTEMWEB%/PatternSkin/header5.gif
   * Set WEBHEADERBGCOLOR = #ffffff

---++ Finalisation

=FINALPREFERENCES= locks site-level preferences that are *not* allowed to be overridden by [[%WEBPREFSTOPIC%]] or user preferences:

   * Set FINALPREFERENCES = ATTACHFILESIZELIMIT, PREVIEWBGIMAGE, WIKITOOLNAME, WIKIHOMEURL, ALLOWROOTCHANGE, DENYROOTCHANGE, USERSWEB, SYSTEMWEB, DOCWEB
